# **Doubly Linked List in JavaScript and TypeScript**

A Doubly Linked List is a type of linked list in which each node contains a data part and two pointers, one pointing to the next node and another pointing to the previous node. This allows traversal in both directions, making it more flexible than a singly linked list.

## **Concept and Use Cases**

**Definition:**
A Doubly Linked List is a linear data structure where each element is a separate object, and each element (or node) contains a data part and two pointers: one pointing to the next node and another to the previous node.

**Common Use Cases:**
- Implementing undo and redo functionalities in applications.
- Navigation systems (e.g., browser history).
- Implementing other complex data structures like dequeues.

## **When to Use**
- When you need to traverse a list bidirectionally.
- When implementing data structures or algorithms that benefit from bidirectional traversal.
- For applications requiring efficient insertion and deletion operations at both ends of the list.

## **Time and Space Complexity**

**Time Complexity:**
- Insertion: O(1)
- Deletion: O(1)
- Access/Search: O(n)

**Space Complexity:**
- O(n) for storing n nodes.
