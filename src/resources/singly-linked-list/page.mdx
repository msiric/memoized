# **Singly Linked List in JavaScript and TypeScript**

A Singly Linked List is a linear data structure where each element, called a node, contains a value and a reference to the next node in the sequence. It is used to efficiently manage collections of data where elements need to be added or removed dynamically.

## **Concept and Use Cases**

**Definition:**
A Singly Linked List is a collection of nodes where each node contains a data part and a reference (or link) to the next node in the sequence. The list terminates with a node pointing to null, indicating the end of the list.

**Common Use Cases:**
- Dynamic memory allocation.
- Implementing stacks and queues.
- Efficiently inserting and deleting nodes.

## **When to Use**
- When you need a data structure with dynamic size.
- For applications that frequently insert and delete elements.
- When managing collections of data where order matters.

## **Time and Space Complexity**

**Time Complexity:**
- Insertion: O(1) if adding at the beginning or end, O(n) if adding at a specific position.
- Deletion: O(1) if deleting the first node, O(n) if deleting a specific node.
- Search: O(n)

**Space Complexity:**
- O(n) for storing n elements.
