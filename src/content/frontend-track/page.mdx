# JavaScript/TypeScript Interview Track

## Core JavaScript Fundamentals

**Description**: Master the fundamental concepts and mechanics of JavaScript
**About**: Build a solid foundation in JavaScript's core concepts, execution model, and language features. Understand how JavaScript works under the hood and handle common interview questions with confidence.

### Lessons

1. **JavaScript Engine & Runtime**

   - V8 Engine Architecture
   - Event Loop & Call Stack
   - Memory Management & Garbage Collection
   - Practice Problems:
     - Implement Event Loop Visualization
     - Debug Memory Leaks
     - Optimize Code for V8

2. **Scope & Closures**

   - Lexical Scope
   - Closure Mechanics
   - Module Pattern
   - Practice Problems:
     - Create Private Counter
     - Implement Memoization
     - Design Module System

3. **this & Prototypes**

   - this Binding Rules
   - Prototype Chain
   - Inheritance Patterns
   - Practice Problems:
     - Implement bind/call/apply
     - Create Inheritance Hierarchy
     - Debug this Context Issues

4. **Asynchronous Programming**

   - Callbacks & Promises
   - async/await
   - Error Handling
   - Practice Problems:
     - Promise Implementation
     - Async Queue Management
     - Error Recovery Patterns

5. **Functions & Execution Context**
   - Function Types & Patterns
   - Execution Context Stack
   - Hoisting & TDZ
   - Practice Problems:
     - Implement Currying
     - Create Function Composition
     - Debug Scope Issues
